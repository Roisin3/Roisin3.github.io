This project was very much a learning experience. I didn't have much trouble coming up with a basic idea for the project. What I wanted was an app that could track player stats of a soccer team. Things like goals, passes, assists, saves, etc. I thought about adding in a games tracker to compare stats of teams and player between games. This seemed a bit too much for this so I saved it for a later date or a future update to the app.

When I finished with the Sintra section coursework I set out to create this app. I ran into many issues caused by typos, those will always haunt me. The biggest hurdles were working out associations, and building the ```player.erb``` view. I reworked associations a few times before I just cut out the need for games and comparing them. The player view is an HTML table. I ran into oneof these in my CLI project but hadn't ever fhad to make one before. So it took a little learning how to create.

Now that this project is done I have a much better idea of how Sinatra and ActiveRecord work and how to build a simple web app. I got some good ideas for future updates to this project as well.
